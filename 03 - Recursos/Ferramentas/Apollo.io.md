# 🔍 Apollo.io

## O que é
Plataforma de **sales intelligence** — banco de dados de empresas e contatos B2B com filtros avançados e enriquecimento.

## Uso no Conectta Hub

- Descoberta de contatos por **domínio** da marca-alvo
- Filtragem por cargo/departamento (marketing, trade, médico)
- Export de lista → validação no [[ZeroBounce]] → import no [[CRM - Hub]]

## Fluxo padrão

1. Busca por empresa → domínio
2. Filtros: cargo contém "marketing", "trade", "medical", "brand"
3. Selecionar 5-15 contatos relevantes
4. Export (CSV)
5. Upload no ZeroBounce para validação
6. Import dos válidos no CRM

## Limitações a lembrar
- Dados podem estar desatualizados (pessoa mudou de empresa)
- Alguns e-mails são inferidos por padrão (`nome.sobrenome@empresa.com`) → **sempre validar**
- Plano gratuito tem cap de créditos
