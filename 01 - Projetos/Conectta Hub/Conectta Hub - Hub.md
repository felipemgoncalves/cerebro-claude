# 🩺 Conectta Hub — Hub

## O que é

Empresa brasileira que **conecta marcas a profissionais de saúde** — OB/GYNs, pediatras, dermatologistas, nutricionistas — através de:

- **Sampling** (amostras e kits)
- **Programas de relacionamento**
- **Canais digitais**
- **Materiais patrocinados / patient guides / checklists brandados**

Modelo **B2B** — cliente é a marca (Bio-Oil, Natura, L'Oréal, Softys/Ontex, CIMED, etc.).
