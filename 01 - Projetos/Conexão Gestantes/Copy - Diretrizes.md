# ✍️ Copy — Diretrizes

Diretrizes consolidadas de copy para **todos os ativos** da Conexão Gestantes (anúncios, landing, emails, WhatsApp).

## Regras absolutas

### ✅ Fazer
- **Tratamento feminino profissional:** "obstetras", "médicas", "profissionais"
- **Benefício concreto** antes de feature
- **CTAs mobile-first** (curtos, claros, botão grande)
- **Prova social** quando possível (número de clientes, depoimentos)
- **Linguagem clínica adequada** (público é médico, não leigo)

### ❌ Não fazer
- **Linguagem de "lançamento" ou "pré-venda"** — produto está no ar
- **CTAs longos ou genéricos** ("saiba mais", "clique aqui")
- **Tom infantilizado** — público é profissional
- **Prometer o que não entrega** (ex.: "triplicar sua consulta")
- **Jargão de infoproduto** ("oferta irresistível", "segredo revelado")

## Banco de CTAs validados

Priorizar:
- "Quero acesso"
- "Ver planos"
- "Começar agora"
- "Quero os materiais"

## Tom de voz

| Eixo | Posição |
|---|---|
| Formal ↔ Informal | **Semi-formal** (profissional, acessível) |
| Técnico ↔ Leigo | **Técnico contido** (sabe que o leitor é médica) |
| Racional ↔ Emocional | **70% racional / 30% emocional** (orgulho profissional, economia de tempo) |
| Curto ↔ Longo | **Curto** para anúncios / **médio** para landing |

## Estrutura de anúncio (Meta Ads)

```
[HOOK visual ou pergunta — 3s]
  ↓
[BENEFÍCIO concreto — 1 frase]
  ↓
[PROVA (opcional) — social ou resultado]
  ↓
[CTA mobile — botão claro]
```

## Gatilhos que funcionam para esse público

- **Economia de tempo** — "horas por semana de volta"
- **Autoridade** — "usado por X obstetras"
- **Padronização** — "padronize suas orientações"
- **Profissionalização** — "atendimento com aparência profissional"

## Gatilhos a evitar

- Medo excessivo ("você está perdendo pacientes")
- Urgência falsa ("só hoje!")
- Promessa exagerada de resultado

## Revisão antes de publicar

- [ ] Tratamento feminino correto?
- [ ] Tem palavra de "lançamento"? → trocar
- [ ] CTA é mobile-friendly?
- [ ] Passa no teste: "uma obstetra leria isso com respeito?"
- [ ] Benefício está claro?
- [ ] Tem prova (quando aplicável)?
