# 📊 Meta Ads Manager

## O que é
Plataforma oficial para **gerenciar anúncios no Facebook, Instagram, Audience Network e Messenger**.

## Uso principal

Tráfego pago do [[Conexão Gestantes - Hub]]. Felipe gerencia direto.

## Princípios operacionais

Consolidados em [[Tráfego Pago - Conhecimento]] e [[Meta Ads - Estratégia]].

## Eventos a garantir

- **PageView** (landing)
- **ViewContent** (engajou com oferta)
- **InitiateCheckout** (foi para Hotmart)
- **Purchase** (comprou)

### API de Conversões (CAPI)
**Altamente recomendada** para resiliência a perdas por ad blockers e iOS tracking.

## Estrutura de nomeação sugerida

```
[Campanha] [Objetivo] [Público] [Data] [Versão]

Ex: CG-Vendas-LL1-abr26-v2
```

## Relatórios importantes

- **Breakdown por placement** (Reels > Feed > Stories?)
- **Breakdown por idade/gênero** (sanity check do público)
- **Criativo ranqueado** (qualidade / engajamento / conversão)
- **Frequência** (saturação)
