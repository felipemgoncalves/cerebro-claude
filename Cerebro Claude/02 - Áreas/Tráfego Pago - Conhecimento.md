# 📢 Tráfego Pago — Conhecimento

Área permanente. Consolida tudo que se aplica a **qualquer** operação de tráfego (não específico de um projeto).

## Princípios gerais

### 1. Frequência antes de ROAS
A frequência alta é o **primeiro sinal** de saturação. Agir antes que o ROAS caia é mais barato que reagir depois.

### 2. Nova campanha > escalar existente
Regra que vale especialmente para contas sensíveis (comprovado no caso [[Meta Ads - Estratégia|Conexão Gestantes]]).

### 3. Uma variável por teste
ABO com conjuntos que variam **uma coisa só** (audiência OU criativo OU público), para leitura limpa.

### 4. Mobile-first é regra, não preferência
85%+ do inventário. Desktop-first é exceção.

### 5. Criativo carrega a campanha
Estrutura boa com criativo médio perde. Estrutura média com criativo excepcional ganha. Priorizar produção de criativo.

## Anatomia de um bom criativo de vídeo

Ver [[Criativos - Vídeo 10 (referência)]] para o caso que virou referência interna.

Regras universais:
- Hook nos **3 primeiros segundos**
- **Sem som = compreensível** (texto em tela obrigatório)
- Uma ideia central — não misture ofertas
- CTA claro e mobile-friendly
- 9:16 prioritário

## Estruturas de campanha

### LL1% (Lookalike 1%)
Primeira escolha para quem tem base de compradores. Audiência "parecida com quem já comprou".

### Aberto (segmentação ampla)
Deixa o algoritmo buscar. Requer volume de dados + criativo forte.

### Interesses
Mais eficaz em nichos onde o interesse é bem definido. Menos estável.

### ABO vs CBO
- **ABO:** controle manual do budget por conjunto → leitura limpa
- **CBO:** algoritmo distribui entre conjuntos → escala, menos leitura

## Métricas críticas

| Métrica | O que indica |
|---|---|
| **CTR** | Se o criativo "para o scroll" |
| **CPC** | Eficiência de impressão para clique |
| **CPA** | Quanto custa converter |
| **ROAS** | Retorno sobre investimento |
| **Frequência** | Saturação de audiência |
| **Hook rate (3s view)** | Eficácia dos primeiros segundos |
| **Thumb-stop ratio** | Retenção inicial |

## Armadilhas comuns

- ❌ Trocar criativo quando o problema é audiência saturada
- ❌ Subir budget bruscamente em campanha vencedora (desestabiliza)
- ❌ Julgar criativo com volume estatístico insuficiente
- ❌ Otimizar para CPC quando objetivo é venda
- ❌ Ignorar comentários (feedback gratuito)
