# ✍️ Copywriting

Área permanente. Regras gerais de copy que se aplicam em **anúncio, landing, e-mail, WhatsApp, LinkedIn**.

## Princípios

### 1. Hook é 80% do trabalho
Se os primeiros 3 segundos (vídeo) ou primeira linha (texto) não param o scroll, o resto nem é lido.

### 2. Benefício > feature
"Economize 5h por semana" > "biblioteca com 200 materiais editáveis".

### 3. Uma ideia por peça
Misturar mensagens dilui todas. **Escolha uma**.

### 4. CTA explícito
"Saiba mais" é fraco. "Quero acesso" é claro. Sempre explícito.

### 5. Prova social quando houver
Número, depoimento, logo. Reduz atrito.

### 6. Palavras do público
Escrever na linguagem que o leitor **já usa**. Pesquisa de copy começa escutando (comentários, DMs, atendimento).

## Estruturas comprovadas

### AIDA
Atenção → Interesse → Desejo → Ação

### PAS
Problema → Agitação → Solução

### 4U (headline)
Útil, Urgente, Único, Ultra-específico

### Hook-Story-Offer
Pegar atenção → construir contexto → fazer oferta

## Tom por canal

| Canal | Tom | Tamanho |
|---|---|---|
| Meta Ads | Direto, benefício-primeiro | Curto |
| Landing | Conversacional, guiado | Médio-longo |
| E-mail prospecção | Profissional, personalizado | Curto |
| WhatsApp | Próximo, informal | Muito curto |
| LinkedIn | Autoridade, insight | Médio |

## Palavras de gatilho (usar com cuidado)

### Funcionam
- Concreto: "R$297", "5h", "350 obstetras"
- Temporal: "este mês", "até sexta", "em 2026"
- Autoridade: "obstetras utilizam", "validado por"
- Profissionalização: "padronize", "organize"

### Desgastadas / evitar
- "Segredo"
- "Revolucionário"
- "Garantido"
- "Oferta irresistível"
- "Últimas vagas" (se não for verdade)

## Checklist antes de publicar

- [ ] Hook forte?
- [ ] Uma ideia central?
- [ ] CTA claro?
- [ ] Prova (se cabível)?
- [ ] Tom adequado ao canal?
- [ ] Alguém do público real leria isso com respeito?
